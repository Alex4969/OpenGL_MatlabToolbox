logiciel pour créer le fichier de police : https://libgdx.com/wiki/tools/hiero
tuto pour utiliser le logiciel : https://www.youtube.com/watch?v=mnIQEQoHHCU 1:20 -> 3:20

 - Dans Font : Choisir une police
 - Dans SampleText : affiche tous les caractère qui seront dans l'image. 
	Choisir au minimum NEHE et ajouter les caractère qui pourrait manquer (exemple : caractères acentués, espace)
 - Dans Effect : Ne rien faire.
 - Dans Padding : Mettre a 3 les valeurs haut, gauche, bas, droite
 - Dans Rendering : Selectionner Glyph cache.
	Faire un reset Cache
	Ici, il faut que le chiffre a coté de page reste à 1 (pour qu'il n'y ai qu'une seule image)
	Pour cela : 
		Il est possible d'augmenter width & Height mais il faut que l'image reste carré
		Il faut reduire la size de la police dans l'ongle Font, plus la size est grande plus le rendu sera jolie
	

La police est prete a être généré.

Pour générer la police faire File -> Save BMFont Files (text)

Terminer !

Attention le logiciel est capricieux et ne fonctionne pas en arrière plan (changer de fenêtre le fait cracher)
